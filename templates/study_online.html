📊 Siguiente paso tras calcular la correlación

1. Confirmar visualmente las correlaciones

Revisar en el gráfico si las correlaciones se reflejan visualmente. Considerar agregar líneas de tendencia para analizar la relación entre variables como Variation y Drawup.

2. Aplicar regresión lineal simple

Usar regresión lineal para explorar relaciones entre pares de variables (por ejemplo, Variation vs Drawup o Amplitude vs Drawdown). Calcular R² y RMSE para medir el ajuste del modelo.

3. Evaluar independencia o multicolinealidad

Si se usan múltiples variables predictoras, verificar si hay correlación alta entre ellas. Esto se puede analizar con la matriz de covarianza o herramientas gráficas como scatterplot matrix.

4. Aplicar prueba de normalidad a los residuos

Luego de una regresión, verificar si los residuos siguen una distribución normal con pruebas como Shapiro-Wilk o mediante histogramas. Esto valida la calidad del modelo.

5. Probar transformaciones si es necesario

Si la relación entre variables no es lineal, aplicar transformaciones como log(x), sqrt(x) o x² para mejorar el ajuste del modelo.

6. Modelos de clasificación o predicción (más adelante)

Considerar usar regresión logística para variables binarias (por ejemplo, si un año terminó en positivo o no), o modelos de series de tiempo si se quiere capturar dependencia temporal.
